# Matriz de Conteúdo — Tâmara Cavalcante

40 ideias cruzando os 5 pilares com 8 formatos. Cada linha é uma pauta pronta para virar post (X), carrossel, Reels ou story. Tom da marca já aplicado. Use como banco — escolha, adapte, produza.

> Como ler: **Pilar** = tema central · **Formato** = como apresentar · **Ângulo** = o conteúdo · **Gancho** = a primeira linha sugerida.
