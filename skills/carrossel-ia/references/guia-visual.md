# Guia Visual

## Formatos padrão

| Plataforma | Formato | Dimensions |
|---|---|---|
| LinkedIn | Infográfico 4:5 | 1080x1350 |
| Instagram | Carrossel 4:5 | 1080x1350 |
| TikTok | Carrossel 9:16 | 1080x1920 |

## Estilos comuns

- Infográfico premium editorial.
- Carrossel educativo.
- Matriz comparativa.
- Funil operacional.
- Checklist visual.
- Fluxo de processo.
- Antes vs depois.
- Sistema/arquitetura visual.

## Regras de texto

- Headline curta e forte.
- Subheadline explica a tese.
- Corpo com blocos curtos.
- Evitar parágrafos longos dentro da imagem.
- Cada slide/imagem deve ter uma função clara.

## Regras de marca

- Usar paleta do cliente em `config.md`.
- Usar assinatura padrão.
- Respeitar posição de assinatura.
- Não inventar logo se o cliente não forneceu.
