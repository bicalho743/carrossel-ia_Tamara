# Como Configurar um Cliente

1. Copie `clientes/exemplo-cliente/`.
2. Renomeie a pasta para o slug do cliente.
3. Edite `config.md`.
4. Coloque referências:
   - `referencias/infograficos/`
   - `referencias/carrosseis/`
   - `referencias/prints-cliente/`
   - `referencias/identidade-visual/`
   - `referencias/exemplos-aprovados/`
5. Crie a pasta da semana:

```text
clientes/[cliente]/carrosseis/semana-01/
```

6. Gere o `.md` final com Briefing + JSONs.

## Convenção de nomes

```text
clientes/[cliente]/carrosseis/semana-XX/[tema]-jsons-imagens.md
```

Exemplo:

```text
clientes/cristian/carrosseis/semana-02/codex-gpt-image-2-jsons-imagens.md
```
