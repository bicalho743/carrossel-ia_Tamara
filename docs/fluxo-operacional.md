# Fluxo Operacional

## 1. Entrada

O usuário informa:

- tema;
- quantidade de imagens;
- plataforma;
- formato;
- referência visual ou pasta de referências;
- feedbacks específicos.

## 2. Leitura

O agente lê:

- `config/marca.md`;
- referências visuais indicadas;
- exemplos aprovados, quando houver.

## 3. Decisão visual

Para cada imagem:

- escolher uma estrutura;
- explicar o uso da estrutura;
- adaptar ao tema;
- definir layout;
- manter branding.

## 4. Escrita do `.md`

O arquivo final deve conter:

- briefing;
- JSONs separados;
- sumário de referências;
- sumário de estruturas visuais.

## 5. Revisão

Antes de finalizar:

- conferir se cada imagem tem JSON próprio;
- conferir `aviso_renderizacao`;
- conferir assinatura e paleta;
- conferir se referências foram usadas como estrutura, não cópia;
- conferir se sumários finais batem com os JSONs.
