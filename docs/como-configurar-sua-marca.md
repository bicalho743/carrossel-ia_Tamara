# Como Configurar Sua Marca

1. Edite `config/marca.md`.
2. Preencha nome, assinatura, tom, paleta e regras visuais.
3. Coloque suas referências:
   - `referencias/infograficos/`
   - `referencias/carrosseis/`
   - `referencias/prints/`
   - `referencias/identidade-visual/`
   - `referencias/exemplos-aprovados/`
4. Crie a pasta da semana:

```text
saidas/semana-01/
```

5. Gere o `.md` final com Briefing + JSONs.

## Convenção de nomes

```text
saidas/semana-XX/[tema]-jsons-imagens.md
```

Exemplo:

```text
saidas/semana-02/codex-gpt-image-2-jsons-imagens.md
```
