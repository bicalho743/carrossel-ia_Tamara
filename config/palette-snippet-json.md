# Snippet de paleta para o bloco `branding.palette` dos JSONs

Substituir a paleta antiga (navy/laranja) por esta nos JSONs gerados:

```json
"branding": {
  "assinatura": "@tamaraorganiza",
  "assinatura_posicao": "bottom_right",
  "palette": {
    "background": "#FAF7F2",
    "surface": "#F5F1EA",
    "surface_dark": "#2C3B2D",
    "primary": "#1E1E1E",
    "accent": "#B8955A",
    "accent_light": "#D4AF7A",
    "muted": "#7A7267",
    "border": "rgba(184,149,90,0.25)"
  },
  "fontes": {
    "titulo": "Cormorant Garamond (serif)",
    "corpo": "Jost (sans-serif)"
  }
}
```

> Lembrete da regra do sistema: nomes de fonte e hex codes são instruções de renderização — NUNCA aparecem como texto visível na imagem.
